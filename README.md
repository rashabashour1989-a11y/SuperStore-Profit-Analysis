# SuperStore-Profit-Analysis
# Project Overview
This project aims to analyze the performance of a retail superstore, identify key factors leading to losser, and build a predictive model to estimate future profits. By leveraging Data Science techniques, we achieved a significant improvement in profit prediction accuracy.
### Key Achievement
* 	**Exploratory Data Analysis (EDA)** : Identified seasonal trends and loss-making categories (Such as Tables).
* 	**Data Cleaning** : Handled missing value and removed outliers to improve model reliability.
* 	**Feature Engineering** : Developed new features like Ship-Days and Month to capture temporal pattern.
* 	**Advanced Modeling** : Implemented an XGBoost Regressor that achieved an accuracy (R²) of 73.02%, significantly outperforming baseline models.
### Task Stack
* 	Language : Paython
* 	Libraries : Pandas, Numpy, Scikit-learn, XGBoost, Matplotlib, seaborn.
* 	Tools : Jupyter Notebook.
### Visualizations :
The project included several insightful charts, such as :
* 	Monthly Sales vs Profit performance.
* 	Category-wise profitability Analysis.
* 	Correlation Heatmaps for feature selection.
### Business Recommendations
1.	Shipping Optimization :Reducing shiping days for high-value orders.
2.	Category Review : Re-evaluating the pricing and discount strategy for the ‘Tables’  category which consistently showed losses.
3.	Seasonal Prep : Increasing inventory during peak sales months identified the analysis.
